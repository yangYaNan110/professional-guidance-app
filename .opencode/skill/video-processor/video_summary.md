# video-summary Skill - 视频摘要生成

## 功能描述

分析视频内容，提取关键知识点，生成视频摘要文本、时间戳标记和剪辑脚本。

## 触发词

总结、摘要、概括、提炼、精华

## 核心能力

1. 分析视频内容，提取关键知识点
2. 生成视频摘要文本
3. 生成时间戳标记
4. 生成剪辑脚本

---

## 摘要内容要求

| 内容类型 | 说明 | 长度 |
|---------|------|------|
| 摘要文本 | 视频核心内容的连贯叙述 | 300-500字 |
| 关键知识点 | 3-5个最重要的知识点 | 每条50字内 |
| 时间戳 | 3-5个重要时间点 | 格式：00:00-说明 |
| 剪辑脚本 | 需要剪辑的片段列表 | 3-5个片段 |

---

## 摘要写作规范

### ✅ 正确做法

- 写成连贯的叙述性文字
- 重点是让用户通过文字了解视频核心内容
- 保持逻辑连贯性

### ❌ 错误做法

- 简单列出关键点
- 碎片化信息堆砌
- 缺少上下文连接

---

## 时间戳标注规则

| 规则 | 说明 |
|-----|------|
| 标注数量 | 3-5个最重要的时间点 |
| 格式 | "时间 - 说明" |
| 选择标准 | 知识点切换、核心概念讲解 |

### 时间戳示例

```
0:00 - 开场介绍
3:15 - 核心概念讲解
8:30 - 案例分析
12:45 - 应用场景
18:00 - 总结展望
```

---

## 关键点提取规则

| 规则 | 说明 |
|-----|------|
| 数量 | 3-5个最重要知识点 |
| 表达 | 简洁、清晰 |
| 逻辑 | 知识点之间有逻辑关系 |

---

## 剪辑脚本规范

### 剪辑目标

| 目标 | 说明 |
|-----|------|
| 时长控制 | 最终输出3-5分钟 |
| 内容精华 | 保留核心知识点 |
| 逻辑完整 | 保持内容连贯性 |

### 片段选择标准

| 标准 | 要求 |
|-----|------|
| 知识点密度 | 选择信息量大的片段 |
| 讲解质量 | 选择讲解清晰的部分 |
| 技术可行性 | 确保片段边界清晰 |
| 内容完整性 | 覆盖主要知识点 |

---

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| bvid | string | 是 | B站视频ID |
| output_duration | int | 否 | 摘要时长（默认300秒） |
| generate_clips | bool | 否 | 是否生成剪辑脚本 |

## 使用示例

```python
from video_summary import VideoSummarizer

summarizer = VideoSummarizer()

# 生成视频摘要
summary = summarizer.summarize(
    bvid="BV1xxx",
    output_duration=300,
    generate_clips=True
)

print(summary.success)
print(summary.summary)
print(summary.key_points)
print(summary.timestamps)

# 生成剪辑脚本
clips = summary.clips
for clip in clips:
    print(f"{clip.start} - {clip.end}: {clip.content}")
```
