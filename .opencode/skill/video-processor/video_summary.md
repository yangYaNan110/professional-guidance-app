# video-summary Skill - 视频摘要生成

## 功能描述

分析视频内容，提取关键知识点，生成视频摘要文本、时间戳标记和剪辑脚本。

## 核心能力

1. 分析视频内容，提取关键知识点
2. 生成视频摘要文本
3. 生成时间戳标记
4. 生成剪辑脚本

---

## 视频摘要生成规范

### 1. 摘要内容要求

- ✅ **摘要要写成连贯的叙述性文字**
- ❌ **不要**简单列出关键点
- ✅ 重点是让用户通过文字就能**大致了解视频的核心内容**
- 📏 长度适中，一般**300-500字**

### 2. 时间戳标记

- ✅ 标注视频中的**重要时间点**
- 📍 格式：`时间 - 说明`

示例：
```
0:00 - 开场介绍
3:15 - 核心概念讲解
8:30 - 案例分析
12:45 - 应用场景
18:00 - 总结展望
```

- 🎯 标注**3-5个最重要的时间点**

### 3. 关键点提取

- ✅ 提取**3-5个最重要的知识点**
- 📝 每个知识点用**简洁的语言概括**
- 🔗 知识点之间要有**逻辑关系**

### 4. 剪辑脚本（如果需要剪辑视频）

- ✅ 标注**起始时间点**
- 📹 每个片段说明其**内容和作用**
- ⏱️ 总时长控制在**3-5分钟**

示例：
```
0:00 - 3:15 - 开场介绍（约3分钟）
8:30 - 12:45 - 案例分析（约4分钟）
18:00 - 20:00 - 总结展望（约2分钟）
```

## 输出格式

### 视频摘要结构

```json
{
  "title": "视频标题",
  "summary": "连贯的叙述性摘要文本...",
  "key_points": [
    "关键知识点1",
    "关键知识点2",
    "关键知识点3"
  ],
  "timestamps": [
    {"time": 0, "label": "开场"},
    {"time": 180, "label": "核心概念"},
    {"time": 450, "label": "案例分析"}
  ],
  "duration": 1200
}
```

### 剪辑脚本结构

```json
{
  "clips": [
    {
      "start": 0,
      "end": 195,
      "content": "开场介绍视频背景和主题",
      "label": "开场"
    },
    {
      "start": 510,
      "end": 765,
      "content": "核心概念详细讲解",
      "label": "核心内容"
    }
  ],
  "total_duration": 1200,
  "output_duration": 300
}
```

## 使用示例

```python
from video_summary import VideoSummarizer

summarizer = VideoSummarizer()

# 生成视频摘要
summary = summarizer.summarize(
    video_info={
        "title": "人工智能导论",
        "description": "本视频介绍了人工智能的起源、发展...",
        "duration": 1200
    },
    transcript="视频字幕内容...",
    output_duration=300
)

print(summary.title)
print(summary.summary)
print(summary.timestamps)
```

## 与其他Skill的配合

- **doc-summarize**：视频摘要的文字内容可进一步使用文档总结Skill进行结构化处理
- **video-clip**：根据剪辑脚本进行视频自动剪辑
- **video-asr**：先使用语音识别获取字幕，再进行分析
