# doc-summarize Skill - 文档智能总结

## 功能描述

对爬取的文档进行智能总结，提取关键信息和知识点，生成结构化的总结内容。

## 核心能力

1. 对文档进行智能总结
2. 提取关键信息和知识点
3. 生成结构化的总结内容
4. 支持多种总结类型

## 总结类型

| 类型 | 说明 |
|------|------|
| `BRIEF` | 简要总结（100字以内） |
| `DETAILED` | 详细总结（分段叙述） |
| `KEY_POINTS` | 关键点提取（列表形式） |
| `TIMELINE` | 时间线梳理（专业介绍专用） |

---

## 专业介绍写作规范（重要）

当生成专业介绍内容时，必须严格遵循以下规则：

### 1. 内容结构要求

- ❌ **不要**分成"溯源、发展、现状、趋势"四个独立模块
- ✅ **要**写成一段**连贯的叙述性内容**
- ✅ 使用Markdown格式，五个段落小标题：
  - **起源与发展**
  - **挫折与争议**
  - **重大突破**
  - **现状与爆发**
  - **未来展望**

### 2. 叙事内容要求

- ✅ **必须**包含**具体年份、人物、国家、事件**
- ✅ 讲述**完整的发展故事**：
  - 从概念提出开始
  - 发展过程中的困难和挫折
  - 重大突破和里程碑
  - 当前的发展态势和成就
  - 未来的发展方向和人才需求

### 3. 正确示例

```
**起源与发展**  
1956年夏天，美国达特茅斯会议上，约翰·麦卡锡、马文·明斯基等科学家首次提出"人工智能"（Artificial Intelligence）概念，标志着AI作为一门学科的诞生。早期AI研究集中在符号主义和联结主义两条路线。

**挫折与争议**  
AI发展历程中经历了两次"寒冬"。1970年代，由于计算能力不足和资金枯竭，AI研究陷入低谷。1980年代，专家系统热潮消退，AI再次进入低谷。

**重大突破**  
2012年是AI发展史上的转折点。深度卷积神经网络在ImageNet竞赛中以85%的准确率夺冠。2017年Google提出Transformer架构，为大语言模型奠定基础。

**现状与爆发**  
当前，AI正以前所未有的速度改变世界。GPT-4、Claude等大模型展现出惊人的语言理解和生成能力。中国AI论文发表量、专利申请量均居全球第一。

**未来展望**  
通用人工智能（AGI）、AI for Science、多模态AI是未来最有前景的方向。预计到2030年，全球AI市场规模将突破10万亿美元。
```

### 4. 错误示例（❌）

```
【溯源】
人工智能专业源于20世纪中期...

【发展】
从最初的机器语言编程到高级语言...

【现状】
当前是全球最热门的技术学科之一...

【趋势】
人工智能、量子计算等方向是未来发展重点...
```

（❌ 问题：四个独立模块，内容割裂，没有具体年份和人物）

### 5. 内容质量要求

- 重点是把这个专业**讲清楚、讲透**
- 让用户看到**发展历程和前景**
- 不要泛泛而谈，要有**具体细节**
- 保持叙述的**连贯性和完整性**

### 6. 领域特定内容

| 领域类型 | 强调重点 |
|---------|---------|
| 计算机类 | 技术革命、产业爆发、人才需求 |
| 法学类 | 法治建设、法律人才需求、社会公平 |
| 医学类 | 技术突破、健康产业、公共卫生 |
| 经济管理类 | 数字经济、金融科技、商业创新 |
| 人文社科类 | 社会发展、文化传承、治理现代化 |

## 使用示例

```python
from doc_summarize import DocumentSummarizer, SummaryType

summarizer = DocumentSummarizer()

# 生成专业介绍（使用时间线格式）
result = summarizer.summarize(
    content=raw_text,
    summary_type=SummaryType.TIMELINE,
    domain="专业"
)

print(result.summary)
```

  "keywords": ["人工智能", "机器学习", ...],

}
