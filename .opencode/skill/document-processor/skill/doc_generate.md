# doc-generate Skill - 文档智能生成

## 功能描述

根据主题、摘要或参考资料生成结构化、专业化的文档内容，支持多种文档类型和格式输出。

## 触发词

生成、创建、编写、制作、产出

## 核心能力

1. 根据主题生成专业介绍文档
2. 基于分析结果生成结构化内容
3. 扩展文档内容
4. 保持文档风格一致性

---

## 专业介绍生成规范

### 时间线格式模板

所有专业介绍必须采用**时间线叙述格式**：

```markdown
**起源与发展**  
[专业名称]的源头可追溯至[具体时间]的[具体事件/人物]。[关键里程碑]标志着[专业]进入[新阶段]。

**挫折与争议**  
[专业]发展历程中经历过[具体挫折/争议]。[影响]。[应对措施]。

**重大突破**  
[年份]，[具体事件]是[专业]史上最伟大的[突破/成就]。[后续影响]。

**现状与爆发**  
当前，[专业]正处于[发展态势]。[具体数据/案例]。[代表性成就]。

**未来展望**  
[专业]未来将在[方向]取得[预期成就]。[人才需求]。[职业前景]。
```

## 文档类型支持

| 文档类型 | 说明 | 特点 |
|---------|------|------|
| 专业介绍 | 详细介绍专业发展历程 | 时间线格式 |
| 行业分析 | 分析行业现状和趋势 | 数据支撑 |
| 职业规划 | 就业方向和职业发展 | 实用导向 |
| 报考指南 | 高考报考指导信息 | 信息全面 |

---

## 内容要素要求

| 要素 | 必须包含 | 示例 |
|-----|---------|------|
| 具体时间 | ✓ | "1956年"、"19世纪末" |
| 关键人物 | ✓ | "约翰·麦卡锡"、"亚当·斯密" |
| 重要事件 | ✓ | "达特茅斯会议"、"《国富论》发表" |
| 国家/地区 | ✓ | "美国"、"欧洲"、"中国" |
| 数据支撑 | ✓ | "市场规模5000亿"、"就业率95%" |

## 专业分类模板

| 学科门类 | 侧重点 | 特有内容 |
|---------|-------|---------|
| 工学 | 技术发展 | 关键技术突破、产业应用 |
| 理学 | 理论创新 | 基础理论、科学发现 |
| 医学 | 技术进步 | 重大疗法、医疗技术 |
| 法学 | 制度演变 | 法律制定、重大案例 |
| 经济学 | 市场变化 | 经济理论、市场发展 |
| 文学 | 文化传承 | 代表作品、文化运动 |
| 管理学 | 方法创新 | 管理理论、企业案例 |

---

## 输入参数

| 参数名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| topic | string | 是 | 生成主题 |
| document_type | string | 否 | 文档类型 |
| major_name | string | 否 | 专业名称 |
| category | string | 否 | 学科类别 |
| depth | string | 否 | 深度（brief/standard/comprehensive） |

## 使用示例

```python
from doc_generate import DocumentGenerator

generator = DocumentGenerator()

# 生成专业介绍
result = generator.generate(
    topic="人工智能专业介绍",
    document_type="专业介绍",
    major_name="人工智能",
    category="工学",
    depth="comprehensive"
)

print(result.success)
print(result.content)
print(result.metadata.is_hot_major)
```

## 与其他Skill的配合

- **doc-crawler**：生成文档可使用文档爬取Skill获取参考资料
- **doc-analyze**：生成文档可进一步使用文档分析Skill验证质量
- **doc-summarize**：长文档可使用文档总结Skill生成摘要
