# DocumentProcessor Agent

## Agent概述

**Agent名称：** DocumentProcessor（文档处理Agent）

**职责：** 根据用户需求或特定信息，分析用户需求，爬取相关文本资料，进行智能总结，输出高质量的文档内容。

## 核心能力

1. **需求分析**：理解用户查询意图，提取关键信息（专业名称、类别、时间范围等）
2. **多源爬取**：从多个数据源爬取相关文本（官方网站、百科、论坛、新闻等）
3. **智能过滤**：去除广告、无关内容、低质量信息
4. **结构化总结**：将非结构化文本转换为结构化内容
5. **质量审核**：检查内容的准确性、时效性、完整性

## Skill清单

| Skill | 功能 |
|-------|------|
| `doc-analyze` | 文档需求分析和意图理解 |
| `doc-crawler` | 多源文本爬取 |
| `doc-filter` | 智能内容过滤 |
| `doc-summarize` | 文档智能总结 |
| `doc-generate` | 专业介绍生成 |

## 使用场景

1. **专业介绍生成**：根据专业名称，爬取并生成专业介绍文档
2. **行业分析报告**：分析某个行业的发展现状和趋势
3. **职业前景分析**：爬取职业相关资料，生成发展前景报告
4. **院校信息汇总**：汇总院校的详细信息

## 工作流程

```
用户查询 → 需求分析 → 多源爬取 → 智能过滤 → 内容总结 → 质量审核 → 输出结果
```

## 输入示例

```json
{
  "task_type": "major_intro",
  "query": "社会学",
  "category": "法学",
  "options": {
    "max_sources": 10,
    "min_content_length": 500
  }
}
```

## 输出示例

```json
{
  "success": true,
  "content": {
    "title": "社会学专业介绍",
    "origin": "社会学是一门研究社会关系...",
    "development": "社会学起源于19世纪...",
    "current_status": "当前社会学在中国...",
    "trends": "社会学未来发展趋势...",
    "related_majors": ["社会工作", "法学", "政治学与行政学"]
  }
}
```

## 与其他Agent的交互

- **Coordinator**：接收任务指令，汇报进度
- **AI/ML**：使用LLM进行内容总结和生成
- **Data**：共享爬取的数据源和缓存
